# Yavix-installer

Program for installing my linux distro

- **`installer.py`**: Main file of the installer.
- **`*_page.py`**: Pages of the installer

# About pages

| Page Name | File | About |
| :--- | :--- | :--- |
| **WelcomePage** | `welcome_page.py` | Page with the **Start installation** button |
| **LanguagePage** | `language_page.py` | Language and timezone selection |
| **PartitionsPage** | `partitions_page.py` | Opens **GParted** for partitioning |
| **DiskPage** | `disk_page.py` | Disk management page |
| **SettingPage** | `setting_page` | User creation and password setup |
| **BrowserPage** | `browser_page` | Browser selection (Firefox by default) |
| **OfficePage** | `office_page` | Office suite selection |
| **InstallationPage** | `installation_page.py` | System installation process |
